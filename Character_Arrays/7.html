Add 5555

  • Read a very long string of digits (at least 6), then add 5555 to this
    • You may not convert to integer. In any case, very large numbers (e.g. 10999999999999999999) can't convert in any case, as integers have a well-known maximum limit!
  • Input → Output
    • 100000 → 105555
    • 10001234 → 10006789
    • 3001072 → 3006627
    • 10999999999 → 11000005554
    • 999999999999999999999999999999999 → 100000000000000000000000000000554
